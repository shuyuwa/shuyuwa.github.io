安装Hexo并部署到gitee pages

安装git

官网地址

https://git-scm.com/download/win

下载64位便携版

64-bit Git for Windows Portable.

配置环境变量

在cmd中输入git –version

git version 2.35.1.windows.2 代表安装好了

安装node.js

官网地址

https://nodejs.org/en/download/

下载64位便携版

配置环境变量

在cmd中输入node -v

有版本号代表安装好了

使用淘宝镜像源

参数后面添加 –registry=https://registry.npmmirror.com

安装Hexo

使用命令

npm install -g hexo-cli –registry=https://registry.npmmirror.com

提示需要升级

npm install -g npm@8.5.2 –registry=https://registry.npmmirror.com

初始化博客

在blog目录下 hexo init

博客的启动

hexo s

添加博客

在source/_posts/这里放新的博客.md文件
hexo clean 清理
hexo g 生成
hexo s 部署

注意！# 后面要加空格

远程部署

安装hexo-deployer-git插件

npm install -g hexo-deployer-git –registry=https://registry.npmmirror.com

修改blog/_config.yml

最底下: deploy:
type: git
repo: https://gitee.com/leoyr/leoyr.git
branch: master

部署到仓库

设置邮箱和名字
git config –global user.email “”
git config –global user.name “”
hexo d
